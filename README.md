# Computer Science Vocabulary Review

一個可離線使用的電腦科學英文詞彙複習工具，提供單字卡、測驗與錯題複習功能。

## 使用方式

你可以直接使用 [GitHub Pages](https://iu-sys.github.io/computer-science-vocabulary-review/)，或下載此儲存庫後，以瀏覽器開啟 `index.html`。所有學習功能皆在瀏覽器本機執行，不需要安裝套件或連線到外部服務。

## 隱私

學習進度只會儲存在你的瀏覽器本機，不會上傳或傳送到任何伺服器。原始 PDF 僅用於整理詞彙內容，不包含在此儲存庫或網站中。
